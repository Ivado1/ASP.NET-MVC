2)Тестовое задание по SQL

ЗАДАНИЕ: написать параметризованный запрос, возвращающий топ-10 регионов, упорядоченных по алфавиту, в которых продажи самого покупаемого товара имеют следующие характеристики:
1)	совокупный объем продаж не менее X единиц;
2)	сумма продаж не менее Y рублей;
3)	число уникальных покупателей не менее Z человек;
Для примера: X=2000, Y=50000000, Z=400.
При наличии нескольких товаров - выбрать тот, у которого сумма продаж наибольшая.
Вывести следующие поля:
●	Название региона;
●	Название товара
●	Количество единиц товара
●	Количество человек
●	Общая сумма
